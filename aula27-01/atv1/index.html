Tudo sobre Tags

Tags HTML são marcações que estruturam um arquivo HTML. Elas são a base de qualquer projeto, como se fossem a sua estrutura óssea

As tags HTML geralmente possuem abertura e fechamento, como no caso de tags de paragrafos. Contudo existem outras que são 'auto fechaveis' como no caso da tag de imagem
